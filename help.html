INSTRUCTIONS

  1. This is two player game where we will have boxes numbered from 1-5, for both players.
  2. Each player will generate a random number from 1-5 and it will be displayed for both players.
  3. As per the number generated, a human will be constructed (step-bystep starting from the head) in its respective row
  4. If the generated number gets repeated in the future turns for the same player the next part of the body will be added to the human of the respective box
  5. The player constructing the full human in all the boxes wins
